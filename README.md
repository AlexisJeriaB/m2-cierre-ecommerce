# Proyecto de finalizacion Modulo N° 2 - Sitio E-Commerce

Finalziacion de proyecto modulo 2, esquema base trabjado en clases.


## Principales Secciones de la página

- Cuenta con una barra de navegación.
- Cuenta con una galería de productos.
- Listado de productos en tarjetas
- Cuenta con links para navegar en la pagina
- Cuenta con botones para Comparar
- Carrito de compras
- Cuenta con un Footer
- Cuenta con una pagina de descripcion del producto

### Tecnologias utilizadas

Se utilizó HTML, CSS, Bootstrap v5.3, JQuery 3.7, JavaScript
